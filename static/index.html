The lightweight, open-source Hotjar alternative.

Features

DOM-Based Replay

Uses MutationObservers to record exact UI changes rather than heavy video, keeping payloads tiny and replays crystal clear.

Privacy by Design

Client-side masking of sensitive PII. Automatically redacts passwords and credit card fields before data leaves the browser.

Geo-Analytics

Real-time event processing with IP-to-Country mapping powered by high-performance Rust lookups.

The Stack

Backend: Rust & Axum

An incredibly fast ingestion engine designed to handle thousands of concurrent event streams with minimal CPU overhead.

Axum Tokio Serde

Edge: Mutation Tracker

A vanilla JS tracking script (less than 10KB) that captures scrolls, clicks, and DOM mutations without slowing down your site.

ES6+ MutationObserver MsgPack

Storage: SQLite (WAL)

Simple, portable, and surprisingly robust. Uses Write-Ahead Logging to ensure smooth concurrent reads and writes.

SQLx WAL Mode Migrations
